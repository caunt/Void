**A quick look at what the Void proxy already supports and what's in development.**

## Terms
<Badge text="Proxying" variant="tip" /> - lets you play through the proxy.  
<Badge text="Redirects" variant="tip" /> - move players between servers.  
<Badge text="API" variant="tip" /> - exposes the Services API to plugins (Chat, [**Commands**](/docs/developing-plugins/commands), [**Events**](/docs/developing-plugins/events/listening-to-events), Content, [**Packets**](/docs/developing-plugins/network/packets)).  
<Badge text="WIP" variant="tip" /> - marks a feature still under development.

:::tip[Online (Private Key)]
Online (Private Key) is a new [**forwarding**](/docs/forwardings/online) method.
It lets you connect to servers using online mode.
The server securely shares its private key so the proxy can expose an API to plugins.
Without the key you can still play, but plugin features are limited.
:::
