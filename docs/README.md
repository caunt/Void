# Void Proxy Documentation

The official documentation for Void Proxy is available at [**https://void.caunt.world**](https://void.caunt.world).

## Documentation Structure

This directory contains two documentation systems:

### Astro Documentation (Active)
- **Location:** `astro/`
- **Status:** Active and deployed to [**void.caunt.world**](https://void.caunt.world)
- **Technology:** Built with Astro and Starlight
- **Purpose:** Official user-facing documentation

See the [**Astro README**](astro/README.md) for more information about contributing to the documentation.

### DocFX Documentation (Testing Only)
- **Location:** `docfx/`
- **Status:** For testing purposes only
- **Purpose:** Used to track and test a DocFX bug in CI/CD
- **Note:** This is not the official documentation
